+ 树莓派 GPIO 0 与 Arduino PIN 21 相连，做好共地

## 调试方法

+ pi 用户目录下输入 ./TaiChi 进入调试器，按下 Ctrl + Z Arduino 开始执行程序

+ 终端内显示 Arduino 发送的调试信息

+ 开始或继续程序执行：Ctrl + Z

+ 暂停程序执行：Ctrl + C
  
+ 重置程序：Ctrl + \
